## 📚 Repository Structure

### 🧩 **Module 1 - Java Basics**

**Java** is one of the most popular and widely used programming languages, known for its ability to develop **robust, secure, and cross-platform** applications.

In this module, I cover the **fundamentals of the language**, including:

* Installation of **JDK** and **NetBeans**
* Basic structure of a Java program
* Data types, variables, and operators
* Conditional and loop structures
* Functions and code modularization
* Input and output handling
* Practical exercises and challenges

📺 **Base Course:** [Curso em Vídeo – Java Básico (Gustavo Guanabara)](https://www.cursoemvideo.com/curso/java-basico/)

---

### 🧠 **Module 2 - Object-Oriented Programming (OOP)**

**Object-Oriented Programming** is at the core of Java.
In this module, I dive deeper into the key concepts that make Java a powerful language for building scalable and well-structured systems.

Topics covered:

* What OOP is and its benefits
* **Classes, objects, and methods**
* **Encapsulation** and visibility (`public`, `private`, `protected`)
* **Special methods** (`getter`, `setter`, `constructor`)
* **Aggregation and composition** between objects
* **Inheritance and polymorphism** (including overloading and overriding)
* Practical exercises and a **final project**

📺 **Base Course:** [Curso em Vídeo – Java POO (Gustavo Guanabara)](https://www.cursoemvideo.com/curso/java-poo/)
